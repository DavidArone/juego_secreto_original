# 🎮 Juego del Número Secreto

Este es el proyecto interactivo desarrollado como parte del curso de Lógica de Programación de **Alura Latam** y Oracle Next Education. Consiste en una aplicación web dinámica donde el usuario intenta adivinar un número generado aleatoriamente por el sistema, ofreciendo retroalimentación en tiempo real.

---

## 🚀 Características

- **Generación Aleatoria:** Números secretos aleatorios con rangos controlables (1 al 50).
- **Retroalimentación Dinámica:** Indica de forma escrita si el número ingresado es mayor o menor al número secreto.
- **Contador de Intentos:** Muestra cuántas veces ha fallado el usuario antes de acertar.
- **Interfaz Interactiva:** Deshabilita y habilita los botones según el estado de la partida actual.
- **Estructura Limpia:** Modulación correcta de funciones JavaScript asociadas a clases e IDs del DOM en HTML.

---

## 🛠️ Tecnologías Utilizadas

- **HTML5:** Estructuración de la interfaz, botones y cajas de texto.
- **CSS3:** Diseño responsivo y estilización visual de los contenedores mediante el archivo `style.css`.
- **JavaScript (ES6):** Manipulación dinámica del DOM (`document.querySelector`), funciones de juego y generación algorítmica.

---

## 📦 Estructura del Repositorio

El proyecto contiene la siguiente estructura de directorios y archivos:

```text
├── img/                # Recursos gráficos e ilustraciones del juego
├── app.js              # Lógica principal, variables y control del juego
├── index.html          # Interfaz de usuario estructurada
└── style.css           # Estilos estéticos de los elementos visuales
```

## 🔧 Instalación y Ejecución Local

Para probar el proyecto de manera local, sigue los pasos de Git descritos a continuación:

1. Clona este repositorio en tu máquina local:
   ```bash
   git clone https://github.com
   ```
2. Accede al directorio del proyecto:
   ```bash
   cd nombre-de-tu-repositorio
   ```
3. Ejecuta el juego abriendo el archivo `index.html` en cualquier navegador web moderno, o utiliza la extensión **Live Server** en VS Code.
